# Deepchem_Practice
Simple fully connected neural network models estimating drug efficacy against HIV and drug water solubility.  
Based on molecular fingerprint featurization by Deepchem library.  

## Environment
- Python = 3.7
- Deepchem = 2.6.0
- Tensorflow = 2.2.0
